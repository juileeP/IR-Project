IR-Project
==========

INFORMATION RETRIEVAL FOR TREC MEDICAL RECORDS

Author: Juilee Patankar

1. Developed a code to enhance free-text searching of de-identified clinical reports, for particular queries.
2. Aim is to get the most relevant reports for a clinical query.
3. Indexed the documents, for faster searching.
4. Carried out pre-processing or query expansion of data, for better medical relevance using PubMed, Mesh Browser and Wikipedia.
5. Used a modified version of TrecDocIterator in Java for querying and parsing the Medical records.
